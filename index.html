⚙️ Setup Supabase

Paste your Supabase project URL and anon key. You can find these at
supabase.com/dashboard → Project Settings → API

📋 SHOW REQUIRED SQL SETUP ▼

Run this in Supabase SQL Editor → New Query:

-- 1. Songs table CREATE TABLE songs ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, name TEXT NOT NULL, artist TEXT, bpm INTEGER, difficulty TEXT DEFAULT 'Normal', description TEXT, file_path TEXT NOT NULL, file_size BIGINT, cover_emoji TEXT DEFAULT '🎵', created_at TIMESTAMPTZ DEFAULT NOW(), updated_at TIMESTAMPTZ DEFAULT NOW() ); -- 2. Scores / Leaderboard table CREATE TABLE scores ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, player_name TEXT NOT NULL, player_id TEXT, song_id UUID REFERENCES songs(id) ON DELETE CASCADE, song_name TEXT, score INTEGER NOT NULL, combo INTEGER DEFAULT 0, accuracy DECIMAL(5,2), difficulty TEXT, played_at TIMESTAMPTZ DEFAULT NOW() ); -- 3. Storage bucket for map files INSERT INTO storage.buckets (id, name, public) VALUES ('maps', 'maps', true); -- 4. Storage policy - allow public read CREATE POLICY "Public read maps" ON storage.objects FOR SELECT USING (bucket_id = 'maps'); -- 5. Allow authenticated upload CREATE POLICY "Authenticated upload maps" ON storage.objects FOR INSERT WITH CHECK (bucket_id = 'maps' AND auth.role() = 'authenticated');
